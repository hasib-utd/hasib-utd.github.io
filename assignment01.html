Q4. What Problems Do You Encounter When Working with the Dataset?

Several issues arise when working with the TEDS 2016 dataset:

  • Missing values: Several variables contain a large number of NA values, particularly vote-related variables such as votetsai, votekmt, and income-related variables.
  • Labelled variables: The dataset is imported from Stata (.dta), so many variables come in as labelled class rather than standard R factor or numeric types, requiring conversion before analysis.
  • Categorical variables stored as numeric: Variables like Tondu, Party, Sex, and Edu are coded as integers but represent categorical categories, which can lead to incorrect analysis if treated as continuous.
  • Skewed distributions: Some variables such as income and age may have skewed distributions that need to be examined before modeling.
Code 
# Check missing values across all variables
TEDS_2016 %>%

Q5. How to Deal with Missing Values?

There are several strategies for handling missing values depending on context:

  • Listwise deletion (drop_na()): Remove all rows with any missing values. Simple but can reduce sample size significantly.
  • Variable-wise filtering: Remove only rows where the specific variable of interest is missing, preserving more data for other analyses.
  • Imputation: Replace missing values with the mean, median, or mode for numeric variables, or use multiple imputation for more rigorous analysis.
  • Flagging: Create a binary indicator variable to flag whether a value was missing, which can itself be informative in modeling.

For this analysis, listwise deletion is used for modeling tasks, while missing values are retained for descriptive summaries.

Code 
# Example: drop rows missing votetsai and key predictors
teds_clean <- TEDS_2016 %>%
  dplyr::select(votetsai, Tondu, female, DPP, age, income, edu, Taiwanese, Econ_worse) %>%
  drop_na()

Q6. Exploring the Relationship Between Tondu and Other Variables

To explore the relationship between Tondu (unification-independence preference) and other variables, the following methods are appropriate:

  • Cross-tabulation / frequency tables for categorical variables (e.g., Tondu vs. female, DPP)
  • Boxplots or violin plots for continuous variables (e.g., Tondu vs. age, income, edu)
  • Correlation analysis for numeric relationships
  • ANOVA or chi-square tests for inferential comparisons
Code 
# Convert Tondu to factor with labels

  levels = 1:7,
  labels = c("Unification now", "Status quo, unif. in future",
